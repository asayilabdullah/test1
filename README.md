# ASAYIL — UI/UX Designer Portfolio

A single-page portfolio website built with plain HTML and CSS, showcasing
UI/UX design work with a classic, professional visual style.

## Task

**Brief:** Design a simple front-end interface that reflects/represents you.

**Requirements:**
- Technology: HTML
- Timeline: 2 weeks
- Deliverable: A personal portfolio, uploaded to GitHub and submitted through the platform
- AI tools (ChatGPT / Claude) were allowed for assistance with the design

**How this submission meets the brief:**
This project is a personal portfolio page for ASAYIL, a UI/UX designer.
It was built as a single static HTML file (no frameworks, no build tools)
to keep it simple and easy to review, while still reflecting a considered,
professional design direction — a monogram seal as a personal mark,
a classic serif/sans-serif type pairing, and a warm, editorial color
palette instead of a generic template look.

## Overview

- **Design style:** Classic & professional
- **Fonts:** Cormorant Garamond (headings), Inter (body text)
- **Built with:** HTML5, CSS3 — no frameworks, no build step
- **Sections:** Hero, About, Skills & Tools, Selected Work, Contact

## How to view it

Just open `asayel-portfolio-en.html` in any browser — no installation
or server required.

## Deploying with GitHub Pages

1. Rename `asayel-portfolio-en.html` to `index.html`.
2. Push it to a GitHub repository.
3. Go to **Settings → Pages**, choose the `main` branch and `/root`, then save.
4. Your site will be live at `https://<username>.github.io/<repo-name>/`.
